osf

Experimental golang package for working with Open Screenplay Format 2.0. Open Screenplay Format is an open XML format for screenplays and the native format (when zipped) for Fade In. Two package will include two demonstration command line programs osf2txt which will read a osf file and render plain text in a Fountain like format as well as fadein2txt which will read in a Fade In file and write out plain text in Fountain format.
